# Rebeca Eco Clean - Cleaning Services Website

A professional, responsive website for a cleaning services company built with Astro and Tailwind CSS.

## Features

- Modern, elegant design
- Fully responsive (mobile-first)
- Professional color palette
- Customer reviews slider
- Contact form
- Multiple service pages

## Tech Stack

- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Routing**: Astro file-based routing

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:4321` to view the site.

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
